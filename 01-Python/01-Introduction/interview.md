# Chapter 1 Interview Questions

1. What is Python, and what makes it general-purpose?
2. What is the difference between Python and CPython?
3. Is Python compiled or interpreted? Explain why the simple answer is incomplete.
4. What does dynamic typing mean?
5. Why is indentation significant in Python?
6. What is bytecode?
7. What does the Python Virtual Machine do in CPython?
8. Name two situations where Python may not be the best primary language.
9. Why should new projects use a supported Python 3 release?
10. How would you investigate a beginner's `IndentationError`?

## Strong Answer Pattern

Give a definition, add one precise technical detail, and provide a small example or tradeoff. Avoid claiming that a simplified beginner explanation is the whole execution model.
