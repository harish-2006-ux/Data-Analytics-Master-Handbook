# Chapter 1 Quiz

Choose one answer for each question.

## Questions

1. Which symbol begins a Python comment?
   - A. `//`
   - B. `#`
   - C. `<!--`
   - D. `--`

2. What does `len([4, 8, 15])` return?
   - A. `2`
   - B. `3`
   - C. `4`
   - D. `27`

3. Which value is a Boolean?
   - A. `"True"`
   - B. `1`
   - C. `True`
   - D. `true`

4. What does indentation primarily define?
   - A. Variable types
   - B. Code blocks
   - C. Package versions
   - D. File encoding

5. Which expression calculates an average for `numbers`?
   - A. `sum(numbers) * len(numbers)`
   - B. `sum(numbers) / len(numbers)`
   - C. `len(numbers) / sum(numbers)`
   - D. `average(numbers)`

6. Which tool is commonly used for tabular data manipulation?
   - A. `pandas`
   - B. `turtle`
   - C. `cmath`
   - D. `pickle`

7. What does `2 ** 3` evaluate to?
   - A. `5`
   - B. `6`
   - C. `8`
   - D. `9`

8. Which name follows the usual Python naming convention?
   - A. `AverageSales`
   - B. `average-sales`
   - C. `average_sales`
   - D. `average sales`
