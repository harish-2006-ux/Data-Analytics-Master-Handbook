# Chapter 1 Practice

## Beginner

1. Print your name, target role, and one reason you are learning Python.
2. Store a temperature in a variable and print whether it is below, equal to, or above 20 degrees.
3. Store a score and print `pass` when it is at least 50; otherwise print `review`.

## Intermediate

4. Write a function that receives a list of scores and returns the average.
5. Extend the function to return both the average and the number of passing scores.
6. Make the function reject an empty list with a clear `ValueError`.

## Advanced

7. Build a command-line learning profile using the requirements in `assignment.md`.
8. Add three test cases and explain the expected output for each.

## Suggested Checks

- Test normal values.
- Test a boundary value such as exactly `50`.
- Test invalid or empty input.
- Compare your output with the examples in the chapter.
