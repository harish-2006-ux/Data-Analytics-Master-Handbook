# Chapter 1 Assignments

## Assignment 1 - Python in Your Field

Write a one-page report explaining one real use of Python in a field you care about. Include:

- The problem Python helps solve.
- Two relevant libraries or tools.
- One limitation or tradeoff.
- At least two reputable references.
- Confidence labels for claims that are facts, implementation details, best practices, or opinions.

## Assignment 2 - First Program

Create a program that asks for a learner's name, target role, and weekly study hours. Print a formatted profile and a recommendation for the next learning action.

Requirements:

- Use variables, strings, numbers, and a conditional statement.
- Put the recommendation logic in a function.
- Include Python version, test status, expected output, and dependencies.

## Assignment 3 - Explain the Execution Model

In your own words, explain the difference between Python source code, bytecode, CPython, and the Python Virtual Machine. State which parts are general Python concepts and which parts are CPython implementation details.

## Submission Checklist

- [ ] The program runs without syntax errors.
- [ ] The output is demonstrated with at least two inputs.
- [ ] The code is readable and uses consistent indentation.
- [ ] Sources are included for factual claims.
- [ ] The Python version and dependencies are documented.
