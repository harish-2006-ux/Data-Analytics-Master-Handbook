# Data Analytics Master Handbook

A professional, versioned handbook for learning data analytics from foundations through applied projects.

## Governing Standard

All chapters are written and reviewed against [DAMS v1.0](Handbook/DAMS-v1.0.md). The chapter structure is fixed so the handbook remains consistent as it grows.

## Version 1.0 Scope

- 40 Python chapters
- 35 SQL chapters
- 30 Pandas chapters
- Statistics fundamentals
- Git and GitHub
- Linux basics

## Repository Layout

| Directory | Purpose |
| --- | --- |
| `Handbook/` | Published chapter manuscripts and the handbook standard |
| `Code/` | Runnable code examples and validation scripts |
| `Datasets/` | Small, documented datasets used by examples and projects |
| `Images/` | Book images and supporting media |
| `Diagrams/` | Source files and exports for visual diagrams |
| `Projects/` | Standalone projects and capstones |
| `Exercises/` | Coding exercises and solutions |
| `Interview/` | Interview questions and preparation material |
| `CheatSheets/` | Condensed reference sheets |
| `Revision/` | Review notes and spaced-repetition material |
| `Resources/` | Books, papers, templates, and other references |

The existing numbered learning paths remain the navigation structure for the current content. The new directories separate the book from its supporting assets and runnable material.

## Development Process

`Research -> Outline -> Write -> Examples -> Exercises -> Review -> Improve -> Publish`

Progress is tracked in the [chapter board manifest](Handbook/CHAPTER-TRACKER.md), which mirrors the GitHub Project board.
