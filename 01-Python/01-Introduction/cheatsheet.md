# Chapter 1 Cheat Sheet

| Concept | Remember |
| --- | --- |
| Python | High-level, general-purpose programming language |
| CPython | Common Python implementation |
| Dynamic typing | Values have types; names can be rebound |
| Indentation | Defines blocks of statements |
| Bytecode | Intermediate form used by CPython execution |
| Python Virtual Machine | Executes CPython bytecode |
| `print()` | Displays a value or message |
| `if` | Runs code conditionally |
| `def` | Defines a function |

## First Program Shape

**Code metadata:** Python Version: 3.12+; Status: Tested; Expected Output: Yes; Dependencies: None

```python
def main():
	print("Hello, Python!")


if __name__ == "__main__":
	main()
```

## First Debugging Checks

1. Read the last line of the traceback.
2. Check spelling, quotes, parentheses, and indentation.
3. Confirm the Python version.
4. Reduce the program to the smallest failing example.
