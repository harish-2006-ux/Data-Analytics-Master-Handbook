# Practice Exercises

Complete these exercises in a new Python file. Use `examples.py` only for syntax reference.

## Exercise 1: Personal Profile

Create variables for your name, learning goal, study hours per week, and whether you have written Python before. Print a short profile using f-strings.

## Exercise 2: Temperature Conversion

Store a temperature in Celsius and convert it to Fahrenheit using:

```text
Fahrenheit = Celsius * 9 / 5 + 32
```

Print the result rounded to two decimal places.

## Exercise 3: Sales Summary

Given `sales = [250, 180, 325, 275, 150]`, calculate and print the total, average, minimum, and maximum sales.

## Exercise 4: Target Check

Using the sales list from Exercise 3, print how many days reached a target of `250` or more.

## Exercise 5: Data Quality Check

Given `values = [10, 12, None, 15, None, 20]`, count the missing values and calculate the average of the non-missing values.

## Exercise 6: Debug the Code

Identify and fix the errors in this code:

```python
revenue = 500
cost = "125"
profit = revenue - cost
if profit > 300
    print("Good profit")
```
