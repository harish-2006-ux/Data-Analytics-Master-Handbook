# Interview Preparation

## Common Questions

### 1. What is Python?

Python is a high-level, general-purpose programming language known for readable syntax, dynamic typing, and a large ecosystem of libraries.

### 2. Why is Python popular in data analytics?

It combines simple syntax with mature libraries for data manipulation, numerical computing, visualization, machine learning, databases, and automation.

### 3. What does dynamically typed mean?

The type belongs to the value at runtime, and a name can later refer to a value of another type. Python still enforces type rules when operations are performed.

### 4. What is indentation used for?

Indentation defines code blocks in Python. Consistent indentation is required for statements such as conditions, loops, and functions.

### 5. What is the difference between a script and a notebook?

A script is a source file normally executed from beginning to end. A notebook runs cells interactively and combines code, output, documentation, and visualizations.

### 6. What is an interpreted language?

It is a language executed by an interpreter at runtime. Python implementations may compile source to bytecode internally, but the developer generally runs code through the Python interpreter.

### 7. What is a variable?

A variable is a name bound to an object or value. Assignment uses `=`, as in `total = 100`.

### 8. How would you make a Python analysis reproducible?

Use clear scripts or notebooks, fixed input data, documented dependencies, controlled random seeds where relevant, and a version-control system such as Git.

## Interview Tip

Answer with a definition, a small example, and a practical data-analysis use case. This demonstrates both language knowledge and application judgment.
